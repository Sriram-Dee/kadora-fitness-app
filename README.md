## ‚ú® Features

### Public Website

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dynamic Content**: Hero sections, pricing plans, and testimonials are managed via a global context.
- **Modern UI**: Built with Tailwind CSS and Shadcn/UI components for a premium look.

### Admin Dashboard

- **Secure Access**: Protected routes require authentication (simulated).
- **Dashboard Overview**: Real-time metrics for traffic, revenue, and new members with sparkline charts.
- **Content Management**: Update site text and images instantly.
- **Pricing Manager**: Modify subscription plans and highlighting.
- **Theme Manager**: Global theme switching (e.g., Neon Green, Cyber Blue).

## üì¶ Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Sriram-Dee/kadora-fitness-app.git
   cd kadora-fitness-app
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
